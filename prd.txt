📌 1. 개요
항목	내용
제품명	나만의 청첩장 (MyWeddingCard)
제품 타입	모바일/웹 기반 청첩장 제작 서비스
대상 사용자	예비 부부 및 일반 사용자
목표	사용자가 직접 입력한 정보를 기반으로 고유한 모바일 청첩장을 자동 생성 및 공유할 수 있도록 함

📌 2. 핵심 기능 요약
기능명	설명
회원가입 및 로그인	이메일 기반 회원 시스템
템플릿 선택	기본 제공되는 디자인 중 선택
청첩장 생성	예식 정보, 사진, 메시지 입력 후 자동 생성
고유 URL 생성	/invite/abcd1234 형식의 개인 청첩장 주소
공유 기능	링크 복사, 카카오톡 공유, QR코드 다운로드
방명록 작성	하객이 축하 메시지를 남길 수 있는 기능
관리자 기능	템플릿 관리, 사용자 목록, 통계 확인
유료화 기능	프리미엄 템플릿 결제, 광고 제거 등 옵션

📌 3. 주요 기능 상세
3.1 회원가입 및 로그인
항목	내용
기능 설명	사용자는 이메일 및 비밀번호를 통해 회원가입/로그인
세부 내용	- 이메일 중복 체크
- 비밀번호 2회 입력
- 로그인 유지 토글
- 비밀번호 찾기
필요 화면	로그인, 회원가입, 비밀번호 찾기

3.2 템플릿 선택
항목	내용
기능 설명	사용자는 다양한 템플릿 중 하나를 선택해 청첩장 제작
세부 내용	- 썸네일 리스트 표시
- 미리보기 기능
- 프리미엄 여부 태그
필요 화면	템플릿 목록, 템플릿 미리보기

3.3 청첩장 제작 폼
항목	내용
기능 설명	사용자 입력 정보를 기반으로 청첩장을 자동 생성
세부 내용	- 신랑/신부 이름, 연락처
- 결혼 날짜 및 시간
- 예식장 정보 (이름, 주소, 지도 링크)
- 계좌번호 표시 여부
- 대표 사진 업로드
- 초대 메시지 직접 입력 or 자동 생성
필요 화면	청첩장 생성 폼, 생성 완료 미리보기

3.4 고유 URL 생성 및 배포
항목	내용
기능 설명	생성된 청첩장은 고유 URL로 접근 가능
세부 내용	- https://사이트주소/invite/고유코드 자동 생성
- URL 복사 버튼
- QR 코드 자동 생성 및 다운로드
- 카카오톡 공유 버튼
필요 화면	생성 완료 페이지, 공유 안내 UI

3.5 방명록 기능
항목	내용
기능 설명	하객이 축하 메시지를 입력할 수 있음
세부 내용	- 이름, 메시지 작성
- 생성자(청첩장 주인)가 메시지 삭제 가능
- 욕설 필터링
필요 화면	방명록 보기/작성 화면, 관리자 뷰

3.6 결제 시스템 (프리미엄 전용)
항목	내용
기능 설명	프리미엄 템플릿 사용 시 결제 유도
세부 내용	- 프리미엄 템플릿 선택 시 가격 안내
- 카카오페이, 토스, 카드 결제 연동
- 결제 완료 후 광고 제거 등 기능 활성화
필요 화면	결제 안내 모달, 결제 완료 페이지

3.7 관리자 대시보드
항목	내용
기능 설명	운영자가 전체 데이터 및 유저를 관리
세부 내용	- 회원 수, 생성된 청첩장 수 확인
- 템플릿 업로드/수정/삭제
- 결제 내역 확인
- 통계 대시보드 (방문자 수, 조회 수 등)
필요 화면	관리자 로그인, 통계 대시보드, 템플릿 관리 화면
